Research Paper Abstract
Modern astronomical surveys such as Gaia Data Release 3 provide high-precision measurements of stellar positions, parallaxes, and photometry for over one billion sources. Traditional analyses often rely on predefined selection criteria, potentially overlooking rare or unusual stellar populations. This study applies unsupervised machine learning techniques to a filtered subset of Gaia DR3 data to identify statistical outliers in a physically constructed parameter space including absolute magnitude, color index, distance, and tangential velocity. An Isolation Forest model was used to detect anomalies without prior labeling. The model identifies a small fraction of stars exhibiting extreme kinematic or photometric properties relative to the dominant stellar population. These results demonstrate the potential of data-driven anomaly detection methods to uncover astrophysically interesting objects in large-scale surveys.
Data Source
This study uses data from Gaia Data Release 3 (Gaia DR3), published by the European Space Agency in 2022. The newest version, Gaia Data Release 4, will release in December of 2026, which is why the researcher is using Gaia Data Release 3. Nevertheless, the dataset still proves to be an excellent indicator of millions of different stars in our universe. Gaia DR3 provides high-precision information, such as Parallax, Proper motion, and Apparent magnitudes (G, BP, and RP bands). These measurements enable derivation of physical properties such as distance, absolute magnitude, tangential velocity, and color index. 

The goal of this project is to identify anomalous stellar populations using unsupervised machine learning techniques. Gaia DR3 is particularly suitable because it provides homogeneous, high-precision astrometry, allows transformation from observational measurements to physically meaningful features, and spans a large and diverse stellar population, increasing the likelihood of detecting rare or unusual objects. This dataset enables construction of a multidimensional feature space including Distance (pc), Absolute magnitude (mag), Tangential velocity (km/s), and Color index (BP − RP). These features combine spatial, kinematic, and photometric information, which is ideal for anomaly detection.
